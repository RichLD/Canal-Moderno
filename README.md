# Canal Moderno

## Descripción

Este repositorio básicamente es para crear un dashboard interactivo que se pueda actualizar fácilmente con la idea de poder proveer visibilidad al equipo de supermercados en cuanto al estatus de sus cadenas, ya sea con el performance que han tenido las distintas marcas, los días de inventario; De esta manera se puedan tomar decisiones con mayor facilidad.

## Objetivos

- 📊 Dashboard interactivo y dinámico
- 🔄 Actualización fácil y rápida de datos
- 👁️ Visibilidad completa del estado de las cadenas
- 📈 Análisis de performance de marcas
- 📦 Gestión de inventario
- 🎯 Facilitar la toma de decisiones

## Características Principales

- Visualización en tiempo real del status de cadenas
- Análisis de performance por marca
- Gestión de días de inventario
- Interfaz intuitiva y fácil de usar
